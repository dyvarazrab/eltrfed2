Сайт начал изготовливаться Никитой Шаховым (https://vk.com/nikika232) для нужд проекта "Электротранспортная Федерация" (https://vk.com/eltrfed_main).
В процессе разработки проект приостановил работу, однако разработку сайта решено не приостанавливать.
Движок сайта может быть использован с разрешения владельца. Иное копирование запрещено. Для получения движка можно написать в ЛС ВК, где будут обсуждены все вопросы.

Написан на языке PHP с использованием БД MySQL и языков JS (возможно, нет, хзлол), CSS. Соответственно, для работы сайта необходим сервер с поддержкой PHP и MySQL.

Для изменения отображаемого имени сайта необходимо в файле /include/settings.php заменить значение переменной $sitename.
Для изменения цветов сайта необходимо в файле styles.css в той же папке заменить лишь цвета, находящиеся в классе :root{}. Все цвета сайта заменятся автоматически.
Если цвета не подходят, то можно редактировать классы, но понимая, что менять. Иначе все сломается.
Шапка сайта находится в файле /include/top.php. Оттуда в блоке div с классом top можно убрать лишние ссылки или добавить свои. Опять же, нужно понимать, что делаешь.
